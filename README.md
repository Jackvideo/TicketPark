<h1>TicketPark</h1>
TicketPark是一个基于SpringCloudAlibaba微服务框架，使用Java语言编写的一个电影院购票系统，用户可以在该平台浏览电影信息，购票和评价。

<h2>详细需求：</h2>
1.	可以进行用户注册、登录。
2.	可以进行电影搜索。
3.	可以进行电影浏览。
4.	可以进行电影票购买。
5.	支持电影点赞，评论。
6.	支持查询订单信息。

<h2>功能模块</h2>
配置模块
  实现相关配置集成
网关模块
  实现对访问请求的过滤
用户模块
用户信息的存储、注册登录、电影票购买
电影模块
  查询电影、展示电影信息、点赞和评论模块
订单模块
 查询订单

<h2>技术栈</h2>
后端：SpringBoot、 Mybatis-Plus 、 SpringCloud Alibaba
数据库：Mysql、Redis
前端：Vue
